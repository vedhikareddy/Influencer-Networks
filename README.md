Influencer Networks and Information Cascades on Social Media
Project Overview
This research project explores the structural and behavioral mechanisms that allow influencers to initiate and propagate massive information cascades across modern social media platforms, including X (Twitter), TikTok, YouTube, and Instagram. The goal is to move beyond simple follower counts and understand the actual flow of influence through network topology and user engagement dynamics.

Research Framework: The ICFM Model
The project proposes the Influencer Cascade Flow Model (ICFM), a theoretical and practical framework designed to study how information moves from a single influential node to the masses.

Technical Methodology & SNA Metrics
Using Social Network Analysis (SNA) techniques and the NetworkX library, the project evaluates influence based on several key centrality metrics:

Degree Centrality: Identifying nodes with the highest number of direct connections.

Betweenness Centrality: Locating "bridge" influencers who control the flow of information between different community clusters.

Eigenvector Centrality: Measuring influence not just by how many people you know, but by who you know (quality of connections).

Cascade Trajectory Prediction: Analyzing the structural mechanisms of "viral" content to predict how far a message will spread based on initial engagement.

Key Features
Cross-Platform Analysis: Comparative study of how algorithms on TikTok (interest-based) differ from X (follower-based) in promoting cascades.

Viral Signal Detection: Framework for identifying early signals of content virality using topological network changes.

Information Bottleneck Identification: Identifying nodes that, if removed, would significantly hinder the spread of information.

Technologies Used
